# ai-website

这是一个基于 React + Vite 的交互 Demo 实验室，用来把 Claude Design /
vibe-coding 调研结果拆成一个个可验证的前端能力样例。

## 当前范围

第一阶段先搭好应用框架和路线图，后续每次围绕一个 topic 完成对应 demo：

- ECharts 交互图表
- Spline 3D 嵌入
- Three.js 自定义 3D 场景
- Shadertoy Shader 改造
- Unicorn Studio WebGL 特效嵌入
- Matter.js 物理实验
- Rive 交互二维动画
- Mapbox 交互地图

## 常用命令

```bash
npm run dev
npm test
npm run build
```

topic-by-topic 的实现计划见 `docs/demo-plan.md`。
